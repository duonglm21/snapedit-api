# Changelog

## 1.0.0

Initial release.

- Full coverage of the SnapEdit API: detection, removal, enhance/restore/colorize,
  generation, AI editing, beauty & style, utilities, virtual try-on, video, and files.
- URL / local path / `Blob` / `Buffer` / `Uint8Array` inputs.
- Automatic retries, timeouts, and rate-limit tracking.
- One-call `.run()` helpers with polling for async task endpoints.
- ESM + CommonJS builds with bundled TypeScript declarations.
