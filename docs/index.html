SnapEdit SDK

Official TypeScript / JavaScript SDK for the SnapEdit API — 40+ AI image & video editing models through a single API key: background removal, object erasing, upscaling, restoration, image generation, virtual try-on, and more.

Quick start

Get an API key from the SnapEdit dashboard — keys start with sk-snap-.

import { SnapEdit } from "snapedit-api";

const client = new SnapEdit({ apiKey: process.env.SNAPEDIT_API_KEY });

// Remove the background from an image URL
const result = await client.remove.background({
  input_image: "https://img.snapedit.app/portrait.jpg",
});

console.log(result.data[0].url);

Configuration

const client = new SnapEdit({
  apiKey: "sk-snap-...",        // required
  authScheme: "bearer",          // or "api-key" header
  timeout: 60000,               // per-request, ms
  maxRetries: 2,                 // retries on 429 / 5xx
});

Examples by category

// Removal
await client.remove.background({ input_image: url });
await client.remove.objects({ input_image: url, erase_mode: "super" });
await client.remove.logo({ input_image: url });    // auto-detects

// Enhance / restore / colorize
await client.enhance.image({ input_image: url, zoom_factor: 4 });
await client.enhance.restore({ input_image: oldPhotoUrl });
await client.enhance.colorize({ input_image: bwPhotoUrl });

// Generation
await client.generate.zimage({
  prompt: "a fox in a snowy forest",
  aspect_ratio: "16:9",
});

Async tasks (try-on, video, PDF)

The .run() helper creates the task and polls to completion in one call.

const task = await client.tryOn.run(
  {
    model_image: personUrl,
    cloth_image: shirtUrl,
    cloth_type: "upper",
    hd_mode: true,
  },
  { onProgress: (t) => console.log(t.status, t.progress) }
);

console.log(task.download_url);

Error handling

try {
  await client.remove.background({ input_image: url });
} catch (err) {
  if (err instanceof SnapEditError) {
    console.error(err.status, err.type, err.message);
    if (err.isRateLimit) console.log(`retry after ${err.retryAfter}s`);
  }
}
